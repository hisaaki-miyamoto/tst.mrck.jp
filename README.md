# tst.mrck.jp — 株式会社丸近 公式サイト（テスト環境）

デザイン検討用の**テスト環境（仮）**です。GitHub Pages で公開しています。

- 目的：直営店サイト（華和ゐ 等）とデザインの統一感を持たせるための、公式サイト トップページ改修案の検討
- 本番URL（予定）：https://tst.mrck.jp/
- 注意：**写真・価格・レビューはすべてダミー（イメージ）**です。

## 主な実装
- 明朝体 × 墨×紅の配色（直営店サイトに合わせた統一トーン）
- フェード自動切替のファーストビュー（slick.js相当を自前JSで実装）
- 縦書きキャッチコピー「香住の恵みを、／食卓へ。」
- 現行公式サイトの和紙テクスチャ背景を継承
- スクロール追従ヘッダー（最上部は透過）／上へ戻るボタン
- ダーク環境でも白基調で固定

## 公開の仕組み
- `index.html` … サイト本体（単一ファイル・画像はBase64埋め込み）
- `CNAME` … カスタムドメイン `tst.mrck.jp` 用

> tst.mrck.jp で表示するには、DNS に次のレコードが必要です：
> `tst  CNAME  hisaaki-miyamoto.github.io.`
